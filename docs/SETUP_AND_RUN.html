2. Database Setup (PostgreSQL)

Start PostgreSQL

# macOS (Homebrew)
brew services start postgresql@16

# Verify
pg_isready

Create Databases

createdb mindforge
createdb mindforge_admin
createdb mindforge_teacher
createdb mindforge_parent

Enable UUID Extension

psql -d mindforge        -c 'CREATE EXTENSION IF NOT EXISTS "uuid-ossp";'
psql -d mindforge_admin  -c 'CREATE EXTENSION IF NOT EXISTS "uuid-ossp";'
psql -d mindforge_teacher -c 'CREATE EXTENSION IF NOT EXISTS "uuid-ossp";'
psql -d mindforge_parent  -c 'CREATE EXTENSION IF NOT EXISTS "uuid-ossp";'

3. Backend Setup

Configure Environment

cd apps/student/backend && cp .env.example .env
cd apps/teacher/backend && cp .env.example .env
cd apps/parent/backend  && cp .env.example .env
cd apps/admin/backend   && cp .env.example .env
Update DB_USERNAME and DB_PASSWORD in each .env to match your PostgreSQL credentials.

Build & Run Migrations

# Student (port 3000)
cd apps/student/backend && npm install && npm run build && npm run migration:run

# Teacher (port 3003)
cd apps/teacher/backend && npm install && npm run build && npm run migration:run

# Parent (port 3002)
cd apps/parent/backend && npm install && npm run build && npm run migration:run

# Admin (port 3004)
cd apps/admin/backend && npm install && npm run build && npm run migration:run

Start All Backends (4 terminals)

cd apps/student/backend && npm run start:dev  # port 3000
cd apps/parent/backend  && npm run start:dev  # port 3002
cd apps/teacher/backend && npm run start:dev  # port 3003
cd apps/admin/backend   && npm run start:dev  # port 3004

Verify

curl http://localhost:3000/health     # Student
curl http://localhost:3002/v1/health  # Parent
curl http://localhost:3003/v1/health  # Teacher
curl http://localhost:3004/health     # Admin

4. Mobile App

Install Dependencies

cd apps/mobile
npm install

Run on Android Emulator

cd apps/mobile
npx expo start
# Press 'a' to open on Android emulator

Run on Physical Device

  1. Enable USB Debugging on your phone
  2. Connect via USB
  3. Run: npx expo run:android

Or use Expo Go: scan the QR code from npx expo start. Set your machine's IP:

EXPO_PUBLIC_API_URL=http://YOUR_IP npx expo start

Build Standalone APK

Option A: Local Build
cd apps/mobile
npx expo run:android --variant release

APK: android/app/build/outputs/apk/release/app-release.apk

Option B: EAS Cloud Build
cd apps/mobile
npx eas build --platform android --profile preview

Download link provided after build completes.

7. Troubleshooting

ECONNREFUSED when starting backend

PostgreSQL not running. Run: brew services start postgresql@16

Android emulator can't reach backend

Android emulator uses 10.0.2.2 for host's localhost. This is auto-configured. Ensure backends are running.

Metro bundler errors

Clear cache: cd apps/mobile && npx expo start -c

Module not found

Reinstall: cd apps/mobile && rm -rf node_modules && npm install
