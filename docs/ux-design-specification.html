5. Accessibility Notes

5.1 WCAG 2.1 AA Compliance Target

Category Requirement Implementation
Color Contrast Min 4.5:1 for normal text, 3:1 for large text Test all text/background combinations
Touch Targets Min 44x44px (recommend 48x48px for kiosk) All buttons, form fields, list items
Focus Indicators Visible focus ring on all interactive elements 2px solid outline, high contrast color
Screen Reader Full ARIA labels on all controls aria-label, aria-describedby for errors
Keyboard Navigation Full keyboard operability Tab order logical; Enter submits
Error Identification Errors identified by more than color alone Icon + text + border
Timing User can extend/disable timeout Timeout modal gives option to continue
Text Resize Readable at 200% zoom Responsive layout; no horizontal scroll
